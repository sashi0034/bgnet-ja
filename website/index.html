インターネットソケットを使おう（日本語版）

原文: https://beej.us/guide/bgnet/ （著者: Brian “Beej Jorgensen” Hall）

これはインターネットソケット（いわゆる「ソケットプログラミング」）を使った ネットワークプログラミングの入門ガイドです。ソケット API は Berkeley 発祥ですが、 Unix・Linux・Windows など多くのプラットフォームに移植されています。

API はやや取っつきにくい面もありますが、C または C++ の基礎があれば、できるだけ手間をかけずにネットワークプログラミングの世界へ 飛び込めるはずです。

このサイトは非公式の日本語訳です。内容の正確さには努めていますが、 公式版は上記の原文サイトを参照してください。
