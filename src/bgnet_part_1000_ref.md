## RFC

[i[RFCs]<] [i[References-->FRFCs]<]

[fl[RFCs|https://www.rfc-editor.org/]]——本当の詳細はここだ！ これらは、インターネットで使われる割り当て番号、プログラミング API、プロトコルを記述した文書だ。楽しみのためにいくつかリンクを載せておく。ポップコーンを用意して、考えモードに入ってほしい：

**[flrfc[RFC 1|1]]**
——最初の RFC。インターネットが生まれたばかりの頃の様子と、地から設計されていく過程の洞察が得られる。（この RFC はもちろん完全に obsolete だ！）

[i[UDP]] **[flrfc[RFC 768|768]]**
——User Datagram Protocol（UDP）

[i[IP]] **[flrfc[RFC 791|791]]**
——Internet Protocol（IP）

[i[TCP]] **[flrfc[RFC 793|793]]**
——Transmission Control Protocol（TCP）

[i[telnet]] **[flrfc[RFC 854|854]]**
——Telnet プロトコル

[i[FTP]] **[flrfc[RFC 959|959]]**
——File Transfer Protocol（FTP）

[i[TFTP]] **[flrfc[RFC 1350|1350]]**
——Trivial File Transfer Protocol（TFTP）

[i[IRC]] **[flrfc[RFC 1459|1459]]**
——Internet Relay Chat Protocol（IRC）

**[flrfc[RFC 1918|1918]]**
——プライベートインターネット向けアドレス割り当て

[i[DHCP]] **[flrfc[RFC 2131|2131]]**
——Dynamic Host Configuration Protocol（DHCP）

[i[HTTP protocol]] **[flrfc[RFC 9110|9110]]**
——Hypertext Transfer Protocol（HTTP）

[i[SMTP]] **[flrfc[RFC 2821|2821]]**
——Simple Mail Transfer Protocol（SMTP）

**[flrfc[RFC 3330|3330]]**
——特殊用途 IPv4 アドレス

**[flrfc[RFC 3493|3493]]**
——IPv6 向け基本ソケットインターフェース拡張

**[flrfc[RFC 3542|3542]]**
——IPv6 向け高度ソケット API

**[flrfc[RFC 3849|3849]]**
——ドキュメント用に予約された IPv6 アドレスプレフィックス

[i[XMPP]] **[flrfc[RFC 3920|3920]]**
——Extensible Messaging and Presence Protocol（XMPP）

[i[NNTP]] **[flrfc[RFC 3977|3977]]**
——Network News Transfer Protocol（NNTP）

**[flrfc[RFC 4193|4193]]**
——ユニークローカル IPv6 ユニキャストアドレス

[i[XDR]] **[flrfc[RFC 4506|4506]]**
——External Data Representation Standard（XDR）

IETF には RFC を [fl[検索・閲覧する便利なオンラインツール|https://tools.ietf.org/rfc/]] がある。
